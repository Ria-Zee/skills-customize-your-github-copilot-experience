
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a word-guessing game using Python strings, loops, conditionals, and user input.

## 📝 Tasks

### 🛠️ Word Selection and Display

#### Description
Create a function that randomly selects a secret word from a predefined list and displays blanks for each letter.

#### Requirements
Completed program should:

- Choose a random word from a list of possible words
- Display the current word state with blanks for unguessed letters
- Keep the secret word hidden until it is correctly guessed

### 🛠️ Guessing Logic and Progress Tracking

#### Description
Implement the letter guessing flow, update the displayed progress, and track incorrect guesses.

#### Requirements
Completed program should:

- Accept single-letter guesses from the player
- Reveal all occurrences of correctly guessed letters in the word
- Track wrong guesses and show remaining attempts

### 🛠️ Win/Lose Conditions and Messages

#### Description
Finish the game by checking whether the player wins or loses and display the correct end message.

#### Requirements
Completed program should:

- End the game when the word is fully guessed or attempts run out
- Print a win message when the word is guessed
- Print a lose message showing the secret word when the player runs out of attempts
